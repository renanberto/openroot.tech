# Hermes / Remote Management

Remote command and get-data architecture for device operations.

## Problem

Operational teams often need controlled ways to request actions from remote devices and inspect the result.
That requires command modeling, authorization, auditability and integration with upstream systems.

## What it covers

- command templates
- remote management actions
- get-data flows
- Cognito-based authorization concepts
- device operation patterns
- API integration and response filtering

## Stack

- DynamoDB
- Cognito
- REST APIs
- command templates
- operational integration design

## What it demonstrates

Hermes demonstrates architecture thinking for remote operations: command definition, permissions, payload shape and operational safety.
