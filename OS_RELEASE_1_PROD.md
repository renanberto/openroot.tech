# OpenRoot OS Release 1 / PROD

Primeiro baseline real de producao para openroot.tech.

## Escopo

- Inspector Shell com apenas Profile, Projects, Skills e Contact.
- Recruiter Tour mantido como badge ao lado do status Online.
- Siderail removido.
- Filesystem com arvore, busca, expand/collapse, context menu, icones por tipo, indicador ativo e resize persistente.
- Terminal com foco automatico ao clicar no painel e ao digitar fora de campos de formulario.
- Timeline removida da navegacao principal.
- GitHub Pages pronto para deploy estatico.

## Build

```bash
npm install
npm run rc:check
```
