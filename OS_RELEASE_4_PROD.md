# OpenRoot OS Release 4 / PROD

Interactive portfolio release for openroot.tech.

## Scope

- Project Viewer and Architecture Viewer in the inspector.
- Cleaner inspector shell navigation with primary tabs and professional subtabs.
- Fixed filesystem search caret behavior and separated palette search state.
- Resize handles for shell panels and resizable document boxes.
- Favorites with filesystem stars, context-menu action and local persistence.
- Keyboard shortcuts and terminal commands for interactive navigation.
- Mobile layout improvements for topbar, panels, cards, viewers and statusbar.

## Build

```bash
npm install
npm run rc:check
```
